== Description ==

Plugin will customize WooCommerce checkout phone field


Plugin Features

1- Checkout Phone Field will have placeholder with format +1(xxx) xxx-xxxx

2- Checkout Phone Field will be mask with format +1(xxx) xxx-xxxx

3- Checkout Phone Field will be validated for phone number with format +1(xxx) xxx-xxxx


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'wc-checkout-custom-billing-phone-field' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done.
